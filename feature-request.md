# Request a Feature

We are always looking for ways to improve Smart Product Options and Addons and make it more useful for your WooCommerce store.

If you have an idea for a new feature, a new field type, or an improvement to an existing tool, we want to hear about it!

## How to Submit a Request

Currently, the best way to submit a feature request is through our GitHub repository.

1. Go to the [Smart Product Options and Addons GitHub Issues](https://github.com/wpanchorbay/smart-product-options-addons/issues) page.
2. Click on the **New Issue** button.
3. Select the **Feature Request** template (if available) or simply write a detailed description of what you'd like to see.
4. Explain *why* this feature is important and how it will help your store.

We review all requests and prioritize them based on community feedback and alignment with our product roadmap.

Thank you for helping us make Smart Product Options and Addons better!
