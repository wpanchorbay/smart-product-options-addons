# OptionBay Screenshot Grouping Plan

This plan groups the screenshots from `image.txt` into logical batches. Each batch corresponds to a single page or state, allowing you to capture multiple images in one sitting.

## Group 1: Option Groups List (The Dashboard)
**Location:** WooCommerce -> OptionBay -> Addon Builder (List View)
**Strategy:** Capture the base list first, then trigger interactive elements (hover, popover, modals).

1.  `option-groups-list.png` (Base view)
2.  `option-groups-header.png` (Static highlight)
3.  `option-groups-filters.png` (Static highlight)
4.  `quick-start-step1.png` (Highlight "Add new group")
5.  `option-groups-row-actions.png` (Hover over a row)
6.  `option-groups-columns.png` (Click the ⋮ column picker)
7.  `option-groups-bulk-confirm.png` (Select items -> Bulk Action -> Move to Trash)
8.  `option-groups-inventory-btn.png` (Highlight 📦 icon)
9.  `export-panel.png` (Click "Export" button)
10. `import-panel.png` (Click "Import" button)
11. `import-success-toast.png` (Trigger a successful import)

## Group 5: Field Type Specifics (General & Options)
**Location:** Expanded Field Rows for each type.
**Strategy:** Iterate through every field type to capture their unique inputs (General/Options/Validation).

*   **Choice Fields:** `field-select-*`, `field-radio-*`, `field-checkbox-*`, `field-checkboxes-*`, `field-color-swatch-*`, `field-image-swatch-*`.
*   **Input Fields:** `field-text-*`, `field-textarea-*`, `field-number-*`, `field-email-*`.
*   **Advanced:** `field-file-*`, `field-static-*`.
